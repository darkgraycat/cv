Description

Developed automation scripts for refactoring, GCP log analysis, and data corruption detection.

Project role

Developer

Responsibilities

  • CLI application designed for seamless navigation among web services such as Datadog, GCP, Spinnaker, and GitHub.
  • Autorefactoring script that replaces all API calls with a custom HttpClient wrapper in a legacy PHP project.
  • Script for automatic injection of custom Logger calls, facilitating easier detection of utilized methods in a legacy PHP project.
  • Script to detect possible data corruption and generate spreadsheed(CSV) with results.
  • And many others...

Stack

JavaScript, Bash, Lua, Rust
